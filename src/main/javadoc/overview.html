FormCoach — AI-Powered Exercise Form Coach

FormCoach is a Java/JavaFX desktop application that uses computer-vision pose estimation to give real-time feedback on exercise technique. A Python MediaPipe server analyses webcam frames and streams 33-point skeleton landmarks back to the Java frontend, which scores form quality and counts repetitions for each supported exercise.

Supported Exercises

  • Push-ups — upper-body pushing movement
  • Sit-ups — core flexion movement
  • Squats — lower-body compound movement

Package Overview

PackageResponsibility
{@link com.formcoach} Application entry point ({@link com.formcoach.Main}) and the central navigation hub ({@link com.formcoach.Navigator}) that wires all screens together.
{@link com.formcoach.auth} User authentication: login, registration, PBKDF2 password hashing, SQLite persistence, and the in-memory {@link com.formcoach.auth.AuthSession} singleton.
{@link com.formcoach.camera} Webcam capture and display via the Sarxos WebcamCapture library. {@link com.formcoach.camera.CameraView} grabs frames and exposes a landmark callback used by the pose pipeline.
{@link com.formcoach.chatbot} Floating AI chat assistant powered by the Google Gemini API. Provides exercise and form advice within the app.
{@link com.formcoach.poseanalysis} Core form-scoring engine. {@link com.formcoach.PoseScorer.PoseScorer} computes 7 joint angles from raw landmarks and finds the best-matching reference pose from the CSV dataset ({@link com.formcoach.PoseReference.PoseReference}).
{@link com.formcoach.repcount} Repetition counting via a two-phase angle-threshold state machine ({@link com.formcoach.repcount.RepetitionCounter}). Works for Push-ups, Squats, and Sit-ups without additional configuration.
{@link com.formcoach.textoutputgen} Converts raw pose-comparison data into human-readable feedback strings and a 0–10 form score ({@link com.formcoach.textoutputgen.textoutputgen}).
{@link com.formcoach.runexercise} Live exercise session screen. Orchestrates the camera, pose scorer, rep counter, and feedback text area in a single JavaFX view.
{@link com.formcoach.selection} Paginated exercise-selection screen where users browse and choose an exercise before entering the tutorial or live session.
{@link com.formcoach.videomodal} Pre-session tutorial video player. {@link com.formcoach.videomodal.VideoResolver} maps exercise names to bundled MP4 files; {@link com.formcoach.videomodal.ExerciseVideoView} renders the playback UI.
{@link com.formcoach.historypage} Workout history screen displaying past session records.
{@link com.formcoach.profile} User profile screen for viewing and updating account details.
{@link com.formcoach.landingpage} Marketing / hero landing page shown on first launch.
{@link kaggle} Kaggle dataset integration. Downloads and caches the exercise CSV on first run; provides {@link kaggle.ExerciseDataset} records to other parts of the app.

Architecture

  Main (JavaFX Application)
    +-- Navigator                   <- single hub, swaps scenes on the Stage
         +-- landingpage
         +-- AuthPage               <- login / register
         +-- ExerciseSelectionPage  <- browse exercises
         +-- ExerciseVideoView      <- tutorial video preview
         \-- runexercise            <- live session
              +-- CameraView        <- webcam + PoseDetector bridge
              +-- PoseScorer        <- angle computation + reference matching
              +-- RepetitionCounter <- state-machine rep counting
              \-- textoutputgen     <- form feedback text + score

Python Pose Server

{@link com.formcoach.PoseDetector.PoseDetector} connects to a local Python process (scripts/main.py) over a length-prefixed TCP socket on port 5001. JPEG frames are sent to Python; MediaPipe landmark JSON is returned. {@link com.formcoach.Main} starts and stops the server automatically alongside the JavaFX lifecycle.
