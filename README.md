CAB302 group project repo.



Brief:

Creating a program that records the user performing exercises, such as pushups, sit-ups, and squats, and provides visual and written feedback in close to real-time indicating to the user whether the movement was done correctly.



Requirements:

Record and display video from the user.

Capture and save images from video.

Obtain a measure of movement quality from captured images. A wireframe denoting the correct posture, and text describing the changes the user should make.

Display the movement quality measures. The pose wireframe overlaid onto the user's video input, and text outputs in a separate UI element.

Allow the user to select between multiple exercises to assess.

Provide a help option with explanations of how to use the program.

Provide narration for text outputs.

Allow the user to ask questions of the program. Implement a chatbot that can provide alternative description of how the program works.
