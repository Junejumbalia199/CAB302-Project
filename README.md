CAB302 group project repo.



Brief:

Creating a program that records the user performing exercises, such as pushups, sit-ups, and squats, and provides visual and written feedback in close to real-time indicating to the user whether the movement was done correctly.


Requirements:

Record and display video from the user.

Capture and save images from video.

Obtain a measure of movement quality from captured images. This can be a wireframe denoting the correct posture, and text describing the changes the user should make.

Connect to an API that uses artificial intelligence to produce information about the user's pose in each frame.

Connect to a database that provides an example of "good form" for the exercises.

Display the movement quality measures. That is, display the pose wireframe overlaid onto the user's video input, and provide text outputs in a separate UI element.

Allow the user to select between multiple exercises to assess.

Provide a help option with explanations of how to use the program.

Possibly, allow the user to ask questions of the program. That is, implement a chatbot that can provide alternative description of how the program works. (Backlog feature)

Allow users to create and login with accounts.

Allow the user to control whether or not video is currently being captured.

Count the user's reps so they can focus on form

Provide accessibility options:

Narration for text outputs.

Ability to scale text size.
